# Quizzo App

An Android-based mobile application for a quiz game focused on country flags. Test your knowledge of flags from around the world!

## Features

* **Multiple Choice Questions:** Each question presents a flag and several options for the country name.
* **Score Tracking:** Keep track of your correct answers.
* **User-Friendly Interface:** Simple and intuitive design for an enjoyable gaming experience.

## Technologies Used

* **Language:** Kotlin
* **Platform:** Android
* **Build System:** Gradle

## Setup Instructions

To get a local copy up and running, follow these simple steps.

### Prerequisites

* Android Studio
* An Android device or emulator

### Installation

1.  **Clone the repository:**

```bash
git clone https://github.com/skan652/quizzo-app.git
```

2.  **Open in Android Studio:**
    * Open Android Studio.
    * Select `Open an existing Android Studio project`.
    * Navigate to the cloned `quizzo-app` directory and select it.
3.  **Run the application:**
    * Connect an Android device to your computer or launch an Android emulator.
    * Click on the `Run` button (green play icon) in Android Studio.

## How to Play

1. Launch the Quizzo App on your Android device or emulator.
2. You will be presented with a flag and multiple country name options.
3. Select the correct country name corresponding to the displayed flag.
4. Your score will be updated based on your answers.
5. Continue playing to improve your knowledge of world flags!
